# Contributing to flyneuro-analyzer

We welcome contributions from the community. Please follow these guidelines.

## How to contribute

1. Fork the repository.
2. Create a new branch for your feature or bug fix.
3. Make your changes.
4. Write or update tests as necessary.
5. Ensure your code passes existing tests.
6. Submit a pull request with a clear description of the changes.

## Reporting bugs

If you find a bug, please open an issue on GitHub. Include a minimal reproducible example and describe the expected vs actual behavior.

## Code style

We follow [PEP 8](https://www.python.org/dev/peps/pep-0008/) for Python code. Use black for formatting.
